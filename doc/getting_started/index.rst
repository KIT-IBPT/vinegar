Getting started
===============

.. toctree::
  :maxdepth: 1
  :caption: Contents:

  install
  dhcp
  environment

In order to use Vinegar, you have two go through three steps:

First you have to install the Vinegar Boot server and create a basic server
configuration. This step is described in :ref:`getting_started_install`.

Second, you have to make the Vinegar server known to your clients by using
appropriate DHCP options. This step is described in :ref:`getting_started_dhcp`.

Finally, you have to add the files and configuration data for your environment.
This is described in :ref:`getting_started_environment`.

One you are finished with this Getting started guide, you might want to continue
by learning more about the :ref:`concepts` behind Vinegar.
